Using MathJax for Equations

This site uses MathJax to render mathematical equations. MathJax enhances the accessibility and readability of equations on the web.

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

You can right-click (or tap and hold on mobile) on any equation to access MathJax options. This allows you to:

  • Adjust the display style for better visibility.
  • Enable double-click to zoom.
  • Access accessibility tools for screen readers.
  • Copy equations as LaTeX, MathML, or plain text.
